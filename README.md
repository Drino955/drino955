### 🔥 What I'm working on

**[ChatCrawler](https://t.me/chatcrawler)** — Telegram OSINT platform

Async scraping system that collects user profiles, groups, channels, and messages from Telegram. Includes a full web management panel, Telegram bot, REST API, and monitoring stack.

| | |
|---|---|
| 👤 22.8M users | 📢 1.28M channels |
| 💬 505K groups | 🤖 30K+ bot accounts |

- PostgreSQL + ClickHouse hybrid DB, 30+ tables
- Multi-account orchestration with rate limiting, proxy pool, auto-scaling
- Panel — Next.js + FastAPI: account manager, Telegram Viewer, AI profile gen, warming system
- Prometheus + Grafana + Loki monitoring with trace IDs
